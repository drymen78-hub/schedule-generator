🚀 시작하기

1. 연/월 → 2. 팀원 설정 → 3. 휴무 입력 → 4. 생성·검토 → 5. 확정·내보내기

이 시스템은 야간 세탁운영팀의 월간 근무 스케줄을 자동으로 생성합니다. 팀원별 휴무일 수, 희망 휴무일, 포지션 요건 등을 입력하면 최적의 스케줄을 자동으로 계산합니다.

접속 방법

브라우저에서 앱 URL로 접속합니다. 별도 로그인이나 설치 없이 바로 사용 가능합니다.

데이터 저장 방식

팀원 설정, 생성된 스케줄, 생산량 데이터는 브라우저 로컬스토리지에 자동 저장됩니다. 같은 PC·브라우저로 접속하면 마지막 상태가 유지됩니다.

⚠️ 브라우저 캐시 삭제 또는 시크릿 모드에서는 데이터가 유지되지 않습니다. 팀 전체가 공유하려면 Google Sheets 연동을 설정하세요.

스케줄 확정 및 내보내기

📤 확정 및 내보내기 버튼을 클릭하면:

  1. 스케줄 JSON이 Google Sheets Sched_YYYY_MM 시트에 저장됩니다
  2. 설정(팀원·포지션)이 AppConfig 시트에 저장됩니다
  3. 각 팀의 시각적 스케줄 시트가 생성됩니다:

📊 Google Sheets 연동

Google Sheets와 연동하면 여러 PC에서 설정과 스케줄을 공유할 수 있습니다.

설정 방법

  1. Google Sheets에서 새 스프레드시트를 만듭니다
  2. 상단 메뉴 [확장 프로그램 → Apps Script]를 클릭합니다
  3. 앱의 ⚙️ 설정 → Apps Script 코드 보기에서 코드를 복사합니다
  4. Apps Script 편집기에 코드를 붙여넣고 저장합니다
  5. [배포 → 새 배포]를 클릭합니다
  6. 유형: 웹 앱, 실행 계정: 나, 액세스: 모든 사용자로 설정합니다
  7. 배포 URL을 복사합니다
  8. 앱 ⚙️ 설정에서 URL을 붙여넣고 저장합니다
⚠️ Apps Script를 수정한 경우 반드시 새 배포(버전 업)를 해야 변경사항이 적용됩니다. 기존 URL로 재배포하면 적용되지 않습니다.

❓ 자주 묻는 질문

Q. 스케줄 생성 후 "고립근무일" 경고가 뜹니다
고립근무일은 휴무-근무1일-휴무 패턴으로, 직원 만족도와 업무 효율 측면에서 지양됩니다. 시스템이 자동으로 수리하지만, 휴무일 수가 너무 많거나 희망휴무가 많은 경우 완전히 해결되지 않을 수 있습니다. 재생성을 여러 번 시도하거나, 특정 날짜의 희망휴무를 조정해보세요.
Q. 희망 휴무일이 반영되지 않았습니다
같은 날짜에 희망휴무와 근무지정이 동시에 설정된 경우 근무지정이 우선됩니다. 3단계로 돌아가서 충돌하는 날짜를 확인해보세요. 스케줄 생성 후 콘솔(F12)에서 [Hard충돌] 메시지로 확인할 수 있습니다.
Q. 일부 팀만 재생성하고 싶습니다
4단계에서 "재생성 팀:" 체크박스를 사용하세요. 체크된 팀만 새로 생성되고, 체크 해제된 팀은 기존 스케줄이 유지됩니다. 3단계에서도 동일한 체크박스를 사용할 수 있습니다.
Q. Google Sheets 연동 후 설정이 저장되지 않습니다
Apps Script 배포 URL이 올바른지 확인하세요. URL은 https://script.google.com/macros/s/.../exec 형식이어야 합니다. 또한 배포 시 액세스를 "모든 사용자"로 설정해야 합니다. 설정 창에서 연결 상태가 "✅ 연결됨"으로 표시되는지 확인하세요.
Q. 다른 PC에서 같은 스케줄을 보려면?
Google Sheets 연동을 설정하고 "확정 및 내보내기"로 저장한 후, 다른 PC에서 같은 앱에 접속하고 설정에서 동일한 Apps Script URL을 입력합니다. 그 후 "팀원·설정 불러오기" 또는 "확정 스케줄 불러오기"를 클릭하면 됩니다.
Q. 엑셀 파일과 Google Sheets 시트의 차이는?
엑셀 다운로드는 내 PC에 파일로 저장하는 방식입니다. Google Sheets 내보내기는 클라우드에 팀별 시각적 스케줄 시트를 생성하며, 팀원들과 링크로 공유할 수 있습니다.
Q. 반차가 스케줄에 영향을 줍니까?
반차는 표시 전용입니다. 실제 스케줄 생성(근무/휴무 배분)에는 영향을 주지 않습니다. 담당자가 참고용으로 확인할 수 있도록 분홍색으로 표시됩니다.

🔧 문제 해결

스케줄 생성이 느립니다

최대 50회 반복 생성하므로 팀원 수가 많거나 조건이 복잡할수록 시간이 걸립니다. 생성 진행률이 표시되며 보통 수 초 내에 완료됩니다.

생성 오류가 발생합니다

오류 메시지가 빨간색으로 표시되면:
1. 팀원이 최소 1명 이상인지 확인하세요
2. 포지션이 설정되어 있는지 확인하세요
3. 희망 휴무일이 총 근무일보다 많지 않은지 확인하세요

Google Sheets 저장이 실패합니다

  1. 설정 창에서 Apps Script URL이 올바른지 확인
  2. Apps Script 배포 상태 확인 (액세스: 모든 사용자)
  3. Apps Script 코드를 최신 버전으로 교체 후 새 배포
  4. 브라우저 콘솔(F12)에서 상세 오류 확인

브라우저 캐시 초기화

앱이 비정상적으로 동작하는 경우 브라우저 개발자 도구 (F12) → Application → Local Storage에서 해당 사이트 항목을 삭제하고 새로고침하세요.

⚠️ 로컬스토리지를 삭제하면 저장된 팀원 설정과 스케줄이 모두 초기화됩니다. Google Sheets에 저장된 데이터는 영향받지 않습니다.
